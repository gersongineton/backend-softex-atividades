# Atividade


O desenvolvimento de um projeto de banco de dados passa por fases que são de grande importância para sua implementação. Descreva as etapas desse processo.


1 - A análise de requisitos - Essa é a fase inicial do projeto a ser desenvolvido. Esse é o momento onde se realiza a identificação dos requisitos que constam em um banco de dados. Nessa fase, as partes são ouvidas para se entender as necessidades e elaborar o projeto conforme a regra de negócio estabelecida. 
 
---------------------

2 - O projeto conceitual - De posse das informações coletadas pela análise de requisitos, utiliza-se um modelo de dados conceitual de alto nível, para gerar o Modelo Entidade-Relacionamento a partir delas. 
O Modelo Entidade-Relacionamento representado através de diagramas, com formas geométricas específicas para cada representação.

---------------------

3 - Projeto lógico - É a etapa onde são definidos os aspectos físicos do projeto, mais precisamente, dos modelos entidade-relacionamento. Cada entidade, torna-se uma tabela com seus relacionamentos e suas regras como: tipos de dados e tamanho dos campos que vão armazenar as informações.

---------------------

4 - Projeto físico - Etapa considerada a parte final do projeto. Nela as estruturas de armazenamento, os índices e a organização de arquivos do banco de dados são definidas. Nessa fase ocorre a geração dos scripts de criação das tabelas e a definição do Sistema Gerenciador de Banco de Dados a ser utilizado, de acordo com as necessidades.
